# Sales & Customer Performance Dashboards
### Overview
This project features a comprehensive suite of Tableau dashboards designed to analyze sales trends and customer behavior. It provides actionable insights into revenue growth, regional performance, and key customer metrics to support data-driven decision-making.


### Key Features

Sales Performance Tracking: Visualizes total revenue, profit margins, and sales growth over time to identify high-performing periods.



Customer Segmentation: Analyzes customer demographics and purchasing patterns to highlight top contributors and acquisition trends.


Regional Analysis: Maps sales data across different geographic locations to pinpoint market opportunities and underperforming areas.



Product Insights: Breaks down performance by category and sub-category to show which products drive the most value.


### Tools & Technologies

Tableau: Used for data visualization and dashboard creation.


Data Source: Structured sales and customer datasets (Hyper/Excel/CSV).
